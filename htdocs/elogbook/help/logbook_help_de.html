Wie finde ich die Einträge:

  1. wähle einen Ordner   entsprechend dem Jahr im linken Fenster
  2. dann wähle das Buch   des entsprechenden Monats
  • das so dargestellte Buch   ist aktuelle geöffnet und wird im rechten Fenster dargestellt
  • das linke Fenster wird aktualisiert durch "SHIFT" + "Neu Laden" Knopf (aktualisiert den Zwischenspeicher)

Was muß ich tun, um ein Bild in das Logbuch zu bekommen ?

  1. Mache ein Bild mit einem Grafik-Program z.b DDD oder Snapshot
  2. drucke dieses Bild mit dem Drucker der "mvpttflog" oder "mvpheralog" heißt, als Postscriptfile.
  3. warte einige Sekunden bis das Bild in der Logbuch Datenstruktur ist.
  4. dann überhole die Anzeige durch drücken   im linken Fenster
  5. drücke auf   bildchen neben deinem neuen Bild und erkläre (mindestens im Titelfeld) was es darstellen soll. Fülle die Schlüsselwort-, Ort- und Bedeutung-Felder aus und speicher den Eintrag.
  • Im Moment können alle Benutzer von allen Computern(Sun,Linux,NT) aus allen MVP-Programmen in das Logbuch drucken.
  • ein Beispiel für einen beliebigen Text aus der UNIX-Komandkonsole : echo "my text" | lp -d mvplog

    Wie funktioniert das Ganze ?

    Die Benutzer Seite :

    • Jedes Program erlaubt es Ausgaben im Postscriptformat zu drucken, die zum Logbuch als Einträge umgeleitet werden.
    • Mit einen Standartbrowser (z.B.Netscape) wird das Logbuch dargestellt.
    • ein kleines Program (JAVA-applet)stellt dynamisch die Struktur des Logbuches im linken Fenster dar.

    Die Server Seite :
    • ein kleines Programm (Shell script)im Hintergrund, hört ständig auf die Drucker (mvpttflog, mvpheralog) und bearbeitet die Druckerausgaben weiter.
    • jeder Eintrag wird als Textfile im Unix-Filesystem unter seinem Datum abgespeichert.
    • Der Webserver wird durch einen Apache web server mit XALAN und XERCES realisiert und läuft unter einer tomcat Maschine
    • Die Seiten mit aktuellen Informationen werden durch JAVA servlets erstellt.
    • In XML files werden die Inhalte der Felder (Meta information) per Eintrag gespeichert.
    • Mit XSL files wird die Darstellung und das Layout der Einträge und der Felder auf den Seiten bestimmt.
